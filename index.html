AquaShield Node

An ultra-low-cost, battery-optimized tank overflow and dry-run protector built for extreme field endurance.

ARCHITECTURE & LOGIC

Sub-20µA Active Floor

Stripped down onboard power LEDs and linear regulator circuits to ensure passive power draw remains at a microscopic level.

Upcycled Li-Ion Loop

Powered via a single salvaged 18650 cell paired with a tiny, weatherproof epoxy solar casing for multi-year off-grid runtimes.

Tactile Probe Arrays

Replaces problematic ultrasonic frameworks with standard marine-grade stainless contacts to bypass fluid humidity corruption.

💡 Core Wiring Diagnostics:

  • GND Bus: Ensure a shared, solid common ground between the Pro Mini, the BC547 emitter, and the 18650 negative terminal.
  • Hardware Pull-up: Pin 2 relies on the ATmega328P's internal 20kΩ pull-up resistor. No physical external resistor is required on the probe lines.
  • Flyback Protection: If modifying this design to use a mechanical relay instead of a piezo buzzer, place a 1N4007 diode across the relay coil to clip inductive voltage spikes.

PRODUCTION FIRMWARE

Deploy this lean loop directly to your micro-architecture via the native Arduino IDE platform. Standby consumption rests at <20µA.

aquashield_sleep.ino 
// AquaShield Node - Core Watchdog Sleep Routine
#include <LowPower.h>

const int PIN_PROBE = 2;   // Structural liquid trace
const int PIN_OUTPUT = 5;  // Gate switcher array

void setup() {
  pinMode(PIN_PROBE, INPUT_PULLUP);
  pinMode(PIN_OUTPUT, OUTPUT);
  digitalWrite(PIN_OUTPUT, LOW);
}

void loop() {
  // Check if fluid has bridged the sensor path (Active LOW)
  if (digitalRead(PIN_PROBE) == LOW) {
    digitalWrite(PIN_OUTPUT, HIGH);
    delay(250);
    digitalWrite(PIN_OUTPUT, LOW);
    delay(250);
  } else {
    // Drop execution completely into deep sleep for 8-second cycles
    LowPower.powerDown(SLEEP_8S, ADC_OFF, BOD_OFF);
  }
}
